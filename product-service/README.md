# Welcome to your CDK TypeScript project

This is a project for AWS CDK development with TypeScript.

####Dependencies:

Docker
AWS CLI
AWS CDK

```
$ sudo npm i -g aws-cdk
```

AWS SAM

### Deployment:

Configure aws credentials with (need IAM User)

```
$ aws configure
```

To deploy to AWS:

```
$ cdk deploy --verbose
```

### Destory:

```
$ cdk destory --verbose
```

### Local Development:

```
$ npm i
```

```
$ npm run dev
```
